Sports Booking Portal


A simple front-end sports venue booking experience built with HTML, CSS, and JavaScript. It lets users explore sports, fill a multi-step booking form, complete a payment step, and view reservation history.

Features
Hero section with sports venue discovery
Sport selection cards
Multi-step booking wizard
Payment method selection
Booking confirmation screen
Reservation history and simulation panel
Project Structure
index.html — main page structure
style.css — styling for the portal UI
app.js — interactive booking logic and UI behavior
How to Run
Open index.html in your browser to view the portal.

If you want to use a local server, you can run:

python -m http.server 8000
Then open http://localhost:8000 in your browser.

Notes
This project is a front-end prototype and uses mock booking flow interactions rather than a real backend service.
